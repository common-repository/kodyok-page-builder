Kodyok Page Builder plugin adds drag and drop page builder functionality to the all themes.

== Description ==

https://www.youtube.com/watch?v=2r_gcoCkbIs

Kodyok Page Builder is used via drag and drop method and it allows you to edit on your website design. Thereby, you can see the changes you made in your website instantly. Kodyok buys you time.

With "Sections", you can quickly apply the ready made designs that you need for your website. We continuously improve and increase these sections.

With "Basic Elements", you can use the object that you desire in your design. We continuously improve and increase these elements.

Including sections and elements, your whole design is automatically compatible with mobile interface. Thus, you can view your website on all screens without a trouble.

== Installation ==

1. Install using the WordPress built-in Plugin installer, or Extract the zip file and drop the contents in the `wp-content/plugins/` directory of your WordPress installation.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Pages > Edit
4. Press the 'Edit with Kodyok' button.
5. Now you can use our drag and drop interface.
